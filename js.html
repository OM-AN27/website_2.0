Equality

JavaScript has abstract ( == ) and strict ( === ) equality operators.

== operator will do its best to do the type conversion and then check the equality: 1 == '1'; will return true.

=== operator will not do any type conversion and will only check the equality, if the two values are not the same type it will return false. 1 === '1'; will return false.
